


# Naming Convention


Elements should obey the naming convention:

`SE.DEV##`


# Sections 

`SE` should be in:

    IN: injector
    MG: merger
    LA: Linac
    CA: Cooling section A
    CB: Cooling section B
    CC: Cooling section C    
    TA: Turnaround A
    RL: Return Line
    TB: Turnaround B
    DU: Dump
    
# Devices

`DEV` should be in:

    QUA: Quadrupole magnet
    BEN: Bending magnet
    PIP: Pipe (drift)
    SOL: Solenoid
    GUN: Gun
    MAR: Marker (zero-length)
